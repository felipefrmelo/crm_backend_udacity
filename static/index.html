Welcome to the CRM Backend API

This API provides a RESTful interface for managing customer data. It supports operations such as retrieving customer lists, adding new customers, updating existing records, and deleting customers. The server is built using Go, leveraging both Fiber and Gorilla Mux frameworks for HTTP handling.

Available Endpoints

  • GET /customers – Retrieve all customers
  • GET /customers/:id – Retrieve a specific customer by ID
  • POST /customers – Add a new customer
  • PUT /customers/:id – Update an existing customer's information
  • DELETE /customers/:id – Remove a customer
